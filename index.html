VoIP.ms SMS

iOS Client for Voip.ms SMS

Instructions

Follow the steps below before loging into Voip.ms SMS on IOS

  • Step 1

    Go to voip.ms, log into your account, click on Main Menu then on SOAP/REST Api

  • Step 2

    Scroll down to the API Configuration table and click on Enable/Disable API. Make sure that it is "enabled". Enter 0.0.0.0 in the Enabled IP Addresses and click on Save IP Addresses. Optionally you can add an API Password which would be different than your regular voip.ms password but this is only optional. If you do you create an api password you will need to use that password to log into voip.ms sms
